Boston Housing Price EDA
프로젝트 소개

Boston Housing 데이터셋을 활용하여 집값에 영향을 미치는 요인을 탐색적 데이터 분석(EDA)을 통해 분석한 프로젝트입니다.

데이터의 분포와 이상치를 확인하고, 변수 간 상관관계를 분석하여 집값과 가장 관련성이 높은 변수를 도출하는 것을 목표로 하였습니다.
